Introduction

CSS stands for Cascading Style Sheets. It is used to style HTML pages.

CSS controls colors, spacing, layouts, fonts, and many other visual aspects of a webpage.

        
        body {
          background: white;
            }
CSS Selectors

CSS selectors are used to target HTML elements that you want to style.

They allow developers to select elements by tag name, class, or ID.

    
        p {
          color: blue;
        }
The Box Model

The CSS box model describes how every HTML element is structured.

It consists of the followings;

  • Content
  • Padding
  • Border
  • Margin
  • Margin
box-sizing: border-box;
Flexbox

Flexbox is a CSS layout system used to arrange elements in rows or columns.

It makes it easier to create responsive layouts.

display: flex;
Media Queries

Media queries allow websites to adapt to different screen sizes.

They are commonly used for responsive web design. 

    
        @media (max-width: 600px) {
          body {
            font-size: 14px;
          }
        }
